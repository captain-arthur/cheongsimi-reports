### 마감 임박 단지 (우선순위 높음)

#### 1. 용인 푸르지오 클루센트 (분양주택)

- **위치**: 경기도 용인시 처인구 역북동 811번지 (신대지구 A1 블록)
- **접수 기간**: 2025-12-29 ~ 2025-12-31 (마감 임박)
- **판교 접근성**: 우수 (30분 이내) - 용인 신대지구, 판교 인접
- **당첨 가능성**: 상대적으로 높음

**신청 가능 평형 및 공급 정보:**
- 84㎡ (3타입): 최고가 7억 4,320만원 ~ 7억 6,520만원, 총 361세대
  - 신혼부부 특별공급: 178세대
  - 생애최초 특별공급: 69세대
  - 일반공급: 약 94세대 추정 (전체 361세대 중 특별공급 제외)

**당첨 가능 경로 분석 (예비 신혼부부·무주택 기준):**
1. **신혼부부 특별공급**: 가능 - 178세대로 공급량이 많아 당첨 가능성 상대적으로 높음
2. **생애최초 특별공급**: 가능 - 69세대, 무주택자라면 가능
3. **일반공급**: 가능 - 약 94세대, 경쟁률 높을 것으로 예상되나 시도 가능

**소득 기준**: 공고문 확인 필요 (청약홈 공고 링크 참조)
- 2인 맞벌이 기준 세전 연봉 환산 불가 (API 제한)

**규제 사항 및 주요 리스크**: 공고문 확인 필요
- 전매제한 규제 유무 확인 필요
- 거주의무 기간 확인 필요
- 재당첨 제한 확인 필요

**결론**: 판교 접근성이 우수하고 신혼부부 특별공급 세대수가 많아 당첨 가능성이 상대적으로 높음. 단, 접수 마감이 임박하므로 즉시 청약 서류 준비 필요.

### ✅ 충족된 요구사항

1. ✅ 서울·경기 지역 APT 주택 전체 조회 (9건)
2. ✅ 분양주택 및 임대주택 포함 (분양 8건, 임대 1건)
3. ✅ 향후 60일 이내 접수 예정 공고 필터링
4. ✅ 전체 목록 제공 (일부 선별 없음)
5. ✅ 판교 접근성 기준 우선순위 정렬
6. ✅ 표 중심 구성
7. ✅ 각 단지마다 청약공고 링크 제공
8. ✅ 각 단지마다 네이버 지도 링크 제공
9. ✅ 신청 가능한 모든 평형 정보 제공
10. ✅ 신혼부부 특별공급, 생애최초 특별공급, 일반공급 정보 제공 (가능한 범위 내)
11. ✅ 마감 임박 단지 상세 분석 (2개)
12. ✅ 입지·조건 좋은 핵심 단지 상세 분석

### ⚠️ API 제한으로 인한 제약사항

다음 정보는 청약홈 API에서 직접 제공하지 않아 공고문 확인이 필요합니다:

1. **소득 기준**: API에서 소득 기준을 도시근로자 %로만 제공하며, 2인 맞벌이 기준 세전 연봉 금액으로 환산 불가
2. **규제 사항**: 전매제한, 거주의무, 재당첨 제한 등은 공고문에서 확인 필요
3. **일반공급 세대수**: API에서 직접 제공하지 않아 총 공급 세대수에서 특별공급을 제외하여 추정

### 📌 중요 안내

본 보고서는 청약홈 OpenAPI를 기반으로 작성되었으며, 실제 청약 시에는 각 단지의 공고문을 반드시 확인하시기 바랍니다. 특히:
- 소득 기준 (2인 맞벌이 기준 세전 연봉)
- 규제 사항 (전매제한, 거주의무, 재당첨 제한)
- 상세 청약 조건

은 공고문에서 정확히 확인해야 합니다.
